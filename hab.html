Tecnologias

Lista de tecnologias que estou estudando ou irei

  • HTML

    • É a linguagem de marcação que estrutura e dá sentido aos elementos da página web.

  • CSS

    • É a linguagem de estilo responsável por definir o design, layout e a aparência visual dos elementos.

  • JAVASCRIPT

    • É a linguagem de programação que adiciona interatividade, lógica e comportamento dinâmico às aplicações.

  • REACT

    • É uma biblioteca JavaScript para criar interfaces de usuário baseadas em componentes reutilizáveis.

  • TYPESCRIPT

    • É um superconjunto do JavaScript que adiciona tipagem estática, ajudando a prevenir erros durante o desenvolvimento.

  • JAVA

    • É uma linguagem de programação orientada a objetos, robusta e muito usada para criar sistemas backend e aplicativos corporativos.
